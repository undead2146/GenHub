# Common UI Components

This directory contains shared UI elements, base classes, and the main window/view structures that are fundamental to the GenHub application's user interface.

## Sub-Directories

*   **[ViewModels](GenHub/GenHub/Common/ViewModels/README.md)**: Contains base view model classes and the main application view model.
*   **[Views](GenHub/GenHub/Common/Views/README.md)**: Contains the XAML and code-behind for the main application window and its primary content view.

## Purpose and Design

The components within `/Common` are designed to:
*   Provide a consistent base for other view models in the application.
*   Define the main shell of the application (the main window).
*   Host the primary content area where different features will be displayed.
*   Encourage adherence to the MVVM (Model-View-ViewModel) pattern.
